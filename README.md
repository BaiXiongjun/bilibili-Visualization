# bilibili-Visualization
bilibili视频可视化工具

## 使用grunt

先装环境

    npm install -g grunt-cli
    npm install

监听sass，不要自己写css

    grunt watch

build，生成dist，会对js、css进行压缩

    grunt build

其他命令看gruntfile.js
